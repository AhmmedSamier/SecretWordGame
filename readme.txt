A simple Secret word game developed for DeskTop application using Windows Forms .
The app is intuitive and allows users to guess words,The word to guess is represented by a row of dashes, representing each letter of the word such as Food , animals , and brands.
each time a word is answered incorrectly the another player will start,finaly the game is over and the guesser wins 

DataBase
The Secret word game uses the Database First design and entity framework with SQL Server

Why is this Secret word game awesome? 
Fantastic design
suitable animations
Clean architecture
